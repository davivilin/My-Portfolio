Animal Cell & Plant Cell 3D Model Showcase

Project Overview

Click the buttons above to switch between the two models. This project is a showcase of my 3D modeling skills from when I was in 7th grade. I created both models using Tinkercad.

My goal was to create a low-poly asset, but still accurate and interactive for my school project. You can interact with the model directly in the viewer above by clicking, dragging, and scrolling!
