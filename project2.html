Code.org App Lab Projects

Project Overview

I created both apps using Code.org's App Lab platform. Both are interactive applications designed to help students study and learn more effectively. The first app is for biology; more specifically, for the Biology EOC exam, which is an important exam in Florida that all students must pass in order to graduate high school. The second app is for learning logical fallacies, which are common errors in reasoning that can undermine the logic of an argument. This app is designed to help students identify and understand these fallacies through an interactive quiz.

I created these apps in 9th grade to enhance my coding skills and to provide useful study tools for my peers. Through this project, I learned how to effectively use App Lab's features, including event handling, data storage, and user interface design.
